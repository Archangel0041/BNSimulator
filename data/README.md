# Data Files

This directory contains the game data files from your Lovable project.

## Expected Files

### units.json
Contains 700+ unit definitions with stats, classes, weapons, and ability references.

### abilities.json  
Contains ability/attack definitions with damage formulas, targeting patterns, ranges, and special effects.

### enums.json
Contains game behavior enums that define mechanics like targeting logic, damage calculations, and status effects.

### encounters.json
Contains enemy encounter layouts with grid positions and unit placements for the 4-wave battle system.
